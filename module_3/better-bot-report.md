## What Happened
**Did the fix work?**
The change to make the bot "motivate the user" did succeed. It focused more on hyping the user and situation up rather than complaining.

**What else changed?**
The tone of the bot became very loud as it exclusively spoke in all-caps. It also stopped using emojis for some reason.

**Is it still "your" bot?**
It is successfully still an anime protagonist, just with a different tone. Because of this, it can still be considered Protagonist Bot


## Reflection
**What made you pick THIS problem over the others?**
So that it would complete tasks with less complaining. I chose 'motivating' instead of changing how 'annoying' it was so it would keep its identity.

**How big was the change compared to its impact?**
It was a large difference because it made everything in all caps. Replacing "whines constantly" with "tries to motivate" made an impact.
